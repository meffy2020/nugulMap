# 너굴맵 모바일 (React Native)

## 실행

```bash
cd mobile
npm install
cp .env.example .env
npm run start
```

- Expo QR로 iOS/Android 기기에서 실행
- `EXPO_PUBLIC_API_BASE_URL`를 백엔드 주소로 설정

## 테스트

```bash
npm run test
```

## 앱 화면 구성
- 지도 탭: 현재 위치 기반 지도 + 마커/선택 상세
- 목록 탭: 검색 가능한 흡연구역 리스트
- 북마크 탭: 로컬 저장된 즐겨찾기
- 등록 모달: 신규 흡연구역 제보(주소/좌표/유형/설명)
- 프로필 모달: 토큰 기반 로그인, 내 정보 조회, 내가 등록한 장소 조회/삭제

## 웹 기능 매핑 현황
- 지도 조회/이동/상세: 모바일 구현 완료
- 키워드 검색: 모바일 구현 완료
- 즐겨찾기: 모바일 구현 완료
- 장소 등록: 모바일 구현 완료(토큰 인증 필요)
- 내 프로필/내 등록 장소 관리: 모바일 구현 완료(토큰 인증 필요)
